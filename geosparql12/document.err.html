<title>Core</title>
<p id="core">This clause establishes the <strong>core</strong> requirements class, with IRI <tt>/req/core</tt>, which has a single corresponding conformance class, <strong>core</strong>, with IRI <tt>/conf/core</tt>. This requirements class defines a set of classes and properties for representing geospatial data. The resulting vocabulary — an ontology — can be used to construct SPARQL graph patterns for querying appropriately modeled geospatial data. RDFS and OWL vocabulary have both been used so that the vocabulary can be understood by systems that support only RDFS entailment and by systems that support OWL-based reasoning.</p>
