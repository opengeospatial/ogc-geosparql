# OGC GeoSPARQL

## 1.2 - ISO co-publication

This branch of the [GeoSPARQL version control repository](https://github.com/opengeospatial/ogc-geosparql/) contains supporting artefacts for a version of [GeoSPARQL 1.1](http://www.opengis.net/doc/IS/geosparql/1.1) that is proposed to be published by the [International Organization for Standardization](https://www.iso.org)' [Technical Committee 211 Geographic information/Geomatics](https://www.iso.org/committee/54904.html) as a 19*-series standard.

### Standard Document

The source files for this version of GeoSPARQL are contained not in this repository but within the ISO's Online Standards Development (OSD) Platform and is currently, as of February 2026, in draft form undergoing TC-211 reviews.

When published, it will be linked to from here.

### RDF Resources

The RDF resources for this version of the standard are taken entirely from v1.1 - see the long-term maintained versions at:

* [Geosemantics DWG-maintained GeoSPARQL SWG Resources](https://github.com/opengeospatial/geosemantics-semantic-resources/tree/main/resources/geosparql-swg)

These resources are all published online via the [OGCs Rainbow Platform](https://defs.opengis.net/prez/) where they can be resolved via IRI:

* [GeoSPARQL 1.1 Ontology](http://www.opengis.net/ont/geosparql)
* [GeoSPARQL RDF Shapes Validator](http://www.opengis.net/def/geosparql/validator)
* [Simple Features Geometries Ontology](http://www.opengis.net/ont/sf)
* [GeoSPARQL 1.1 Functions](http://www.opengis.net/def/function/geosparql)
* [GeoSPARQL 1.1 Rules](http://www.opengis.net/def/rule/geosparql)
* [GeoSPARQL 1.1 Requirements and Conformance Class Register](http://www.opengis.net/def/geosparql/reqs)
