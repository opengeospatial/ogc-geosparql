# GeoSPARQL Rules

This version of GeoSPARQL provides several different executable rules files for the GeoSPARQL Rules.

For the rules definitions, see 

* [GeoSPARQL 1.3 Rules](http://www.opengis.net/def/rule/geosparql/1.3)

The executable files are as follows:

* `rules.rifps` - Rule Interchange Format (RIF)
  * As per the W3C's [RIF standard](https://www.w3.org/TR/rif-overview/)
* `rules.sparql` - SPARQL
  * As per the W3C's [SPARQL 1.2 Update standard](https://www.w3.org/TR/sparql12-update/)
* `rules.srl` - SHACL Rules, in the Shapes Rules syntax
  * As per the W3C's [SHACL 1.2 Rules standard](https://www.w3.org/TR/shacl12-rules/)
