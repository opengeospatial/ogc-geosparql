GeoSPARQL 1.1

... an update of GeoSPARQL 1.0 with non-breaking changes

GeoSPARQL 1.1 constitutes an update to GeoSPARQL 1.0 including among others the following changes:

  • Support for spatial measurement properties
  • Support for FeatureCollections and GeometryCollections
  • Graph Validation with SHACL
  • Definition of GeoSPARQL as a W3C Profile
A comprehensive overview of the motivations behind GeoSPARQL 1.1 can be found in the accompanying paper https://doi.org/10.3390/ijgi11020117
